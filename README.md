JS-Fundamentals
Description

This project focuses on learning the fundamentals of JavaScript.
It covers scripting basics and front-end concepts, including variables, data types, operators, control flow, loops, and functions.
The goal is to understand how JavaScript works and how to use it to build dynamic programs.

Learning Objectives

By the end of this project, I will be able to:
Explain why JavaScript is an amazing programming language
Run a JavaScript script using Node.js
Create and use variables and constants
Understand the difference between var, let, and const
Identify all available data types in JavaScript
Use if and if...else statements correctly
Write and use comments effectively

Assign and update variable values

Use while and for loops

Apply break and continue statements

Define and use functions

Understand what happens when a function has no return statement

Work with variable scope

Perform calculations using arithmetic operators

Manipulate objects and arrays

Import and use modules
